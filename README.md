# Agencia Nathaly · Plataforma de Agencia Marketing/RRSS

**Cliente:** Natalia
**Proyecto:** Plataforma SaaS para agencia de marketing/redes sociales
**Modalidad:** Propuesta comercial · 4 fases incrementales
**Confidencialidad:** Datos comerciales del cliente · público controlado

---

## Estructura propuesta

### Dashboard único agencia
- Acceso con roles personalizados por cliente
- Guías generativas con guiones <24 horas
- Reportes automáticos y métricas centralizadas
- Calendario de contenido integrado

### Gestión financiera y operativa
- Margen por cliente y seguimiento de horas
- Estados de cobranza y reportes mensuales
- Portal separado para clientes

### Roles
- Dueño · Ejecutivo · Freelance · Cliente externo

### Stack técnico
- **Frontend**: Next.js
- **Backend/DB**: Supabase
- **Deploy**: AWS o Vercel
- **APIs**: Meta · TikTok · YouTube · Instagram
- **Mobile**: funcional con datos aislados por cliente
